# Blackjack Casino Game

## Description
Simple Blackjack game built with HTML, CSS, and JavaScript.

## How to use

1. Choose betting wager
2. Click deal.
3. Choose between Hit, Stand, and Double
4. Winner is decided between Player value and Dealer value

## Built-With
- Vanilla JavaScript
- SASS
